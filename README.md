# Sales Performance & Target Achievement Dashboard

## Project Overview

This project is an interactive Power BI dashboard developed to analyze business performance across regions, categories, and Regional Managers.

The dashboard helps management monitor:

* Sales Performance
* Profitability
* Target Achievement
* Regional Manager Performance
* Product Returns
* Delivery Delays
* Year-over-Year Growth Analysis

## Business Problem

The organization required a centralized dashboard to track business performance and identify:

* Regions missing sales targets
* Category-wise performance gaps
* Manager performance
* Product return trends
* Delivery delay risks

## Tools Used

* Power BI
* Power Query
* DAX
* Data Modeling
* Data Visualization

## Key KPIs

* Total Sales
* Total Profit
* Total Orders
* Target Achievement %
* Return %
* Average Delivery Days
* Sales Growth %
* Profit Growth %

## Dashboard Pages

### Executive Summary

Provides an overview of overall business performance through KPIs and trend analysis.

### Manager Performance

Analyzes Regional Manager performance and category-wise target achievement.

### Returns & Risk Analysis

Tracks returns, delivery delays, and operational risks.
